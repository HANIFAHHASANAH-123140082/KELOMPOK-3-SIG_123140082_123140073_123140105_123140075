## Deskripsi
Aplikasi WebGIS untuk menampilkan, mengelola, dan menganalisis 
lokasi parkir publik di Kecamatan Ratu Agung, Kota Bengkulu.

## Tech Stack
- Database: PostgreSQL + PostGIS
- Backend: FastAPI (Python)
- Frontend: ReactJS + Leaflet.js
- Deployment: Railway (backend) + Vercel (frontend)
